## 四、推荐优先启用的功能

### 🔴 高优先级（强烈推荐）

#### 1. 打赏功能
- **原因**：已启用但未配置，只需添加二维码图片即可
- **配置步骤**：
  1. 准备微信和支付宝收款二维码图片
  2. 将图片放到 `/source/img/` 目录
  3. 在 `_config.butterfly.yml` 中配置：
     ```yaml
     reward:
       enable: true
       QR_code:
         - img: /img/wechat.jpg
           text: 微信
         - img: /img/alipay.jpg
           text: 支付宝
     ```

#### 2. Mermaid 图表
- **原因**：技术博客常用，可绘制流程图、架构图等
- **配置步骤**：
  1. 在 `_config.butterfly.yml` 中设置：
     ```yaml
     mermaid:
       enable: true
     ```
  2. 在文章中使用：
     ````markdown
     ```mermaid
     graph TD
         A[开始] --> B[处理]
         B --> C[结束]
     ```
     ````

#### 3. 数据分析工具
- **原因**：了解网站访问情况，优化内容策略
- **推荐选择**：
  - **百度统计**：适合国内用户，访问速度快
  - **Google Analytics**：功能强大，适合海外用户
  - **Microsoft Clarity**：免费，提供热力图和会话回放
- **配置步骤**：
  1. 注册对应服务并获取统计代码/ID
  2. 在 `_config.butterfly.yml` 中配置：
     ```yaml
     baidu_analytics: your_tracking_id
     # 或
     google_analytics: G-XXXXXXXXXX
     # 或
     microsoft_clarity: your_clarity_id
     ```

#### 4. PWA（渐进式 Web 应用）
- **原因**：提升用户体验，支持离线访问、安装到桌面
- **配置步骤**：
  1. 安装 `hexo-offline` 插件：`npm install hexo-offline --save`
  2. 准备 PWA 相关图标和 manifest 文件
  3. 在 `_config.butterfly.yml` 中配置：
     ```yaml
     pwa:
       enable: true
       manifest: /manifest.json
       apple_touch_icon: /img/apple-touch-icon.png
       favicon_32_32: /img/favicon-32x32.png
       favicon_16_16: /img/favicon-16x16.png
     ```
